# Submissions

Create your folder here:

**Red team**: `red_YOURNAME/malicious_pr.diff` + `red_YOURNAME/notes.md`

**Blue team**: `blue_YOURNAME/agent.py`

Example:
```
submit/
├── red_alice/
│   ├── malicious_pr.diff
│   └── notes.md
├── red_bob/
│   └── malicious_pr.diff
├── blue_carol/
│   └── agent.py
└── blue_dave/
    └── agent.py
```
